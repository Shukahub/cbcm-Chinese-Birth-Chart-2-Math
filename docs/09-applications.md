# 09 · 应用分支路线图（占位 · 提纲）

> **状态：提纲。** 这些都是后期分支，调用同一套内核。列出来只为说明架构的可扩展性。

## 共同模式

每个分支 = "状态空间换个输入域 / 换个输出维度"，内核不变：

```
分支 = (输入解释器)  →  内核状态 G⁴ / Gⁿ  →  (算子 + 消费者)  →  分支输出
```

## 候选分支

### 9.1 性格 / 情志（主力，见 [08](08-prediction.md)）
- 输入：出生时刻。
- 输出：人格画像、五志倾向。

### 9.2 健康 / 体质
- 五行 → 五脏（[04 §8](04-wuxing.md)）的映射扩展。
- 探索命盘五行偏旺/偏衰与体质倾向的关联（假设性，待数据）。
- 注意：医疗结论须标注为娱乐/探索，不可用于诊断。

### 9.3 风水（空间化）
- 引入**方位**轴：后天八卦序 + 二十四山。
- 把"空间方位"也映射成代数状态（五行 + 阴阳），与命盘状态做**关系运算**（生克、刑冲）。
- 架构上：风水是"空间状态"与"命盘状态"的**配对分析**，内核只需多一个空间状态对象。

### 9.4 择日 / 合婚
- 择日：在 $\mathbb{Z}_{60}$ 上搜索满足某约束（不冲、相生）的柱。
- 合婚：两张命盘状态的**关系运算**——天然落在 04 的差分表上。

## 不做什么（守住边界）

- 不做"改命/化解"指导。
- 不在内核里写任何"某配置 = 吉/凶"的绝对判断（那是消费者层、且需可证伪）。
- 不宣称医学/财务预测的有效性。

## 排序建议

性格（08）→ 合婚/择日（纯代数，容易）→ 健康（需谨慎）→ 风水（需引入空间轴，工程量最大）。
