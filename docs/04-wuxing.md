# 04 · 五行代数

> 本章把五行生克写成代数。最漂亮的结论：**相克 = 相生²**，整个生克体系不过是一个 5 阶循环群 $C_5$。

---

## 0. 传统说法（古法原典）

> 本章先按古人自己的话讲一遍，再做代数翻译（文末小结即对照表）。

- **五行**：金、木、水、火、土，为构成万物的五类基本之气。
- **性情**（《尚书·洪范》）："水曰润下，火曰炎上，木曰曲直，金曰从革，土爰稼穑。"
- **相生**：木生火、火生土、土生金、金生水、水生木。生我者为父母，我生者为子孙。
- **相克**：木克土、土克水、水克火、火克金、金克木。克我者为官鬼，我克者为妻财。
- **五种关系**：生我、我生、克我、我克、比和（同类），共成五类，两两之间必居其一。
- **扩展**：五行配五方（东西中西北）、五季、五脏、五志、五色……（见 §8）。

> 关键观察：古人把相生、相克讲成两套并列的规则。下面代数将揭示——**它们其实是同一个循环的两次幂**。

## 1. 五行集 = Z₅

定义五行集 $E = \mathbb{Z}_5$，名称表

```
WUXING = [木, 火, 土, 金, 水]
          0   1   2   3   4
```

## 2. 相生 = 循环置换 σ

传统相生序：木→火→土→金→水→（回木）。这是一个 5-循环：

$$
\sigma = (\text{木}\ \text{火}\ \text{土}\ \text{金}\ \text{水}), \qquad \sigma(x) = x + 1 \pmod 5
$$

读法：$x$ **生** $\sigma(x)$。即"我生的是我后一位"。

- 木(0) 生 火(1) ✓
- 火(1) 生 土(2) ✓
- 土(2) 生 金(3) ✓
- 金(3) 生 水(4) ✓
- 水(4) 生 木(0) ✓

## 3. 相克 = σ²（关键结论）

传统相克序：木→土，土→水，水→火，火→金，金→（回木）。看 $\sigma^2$：

$$
\sigma^2(x) = x + 2 \pmod 5
$$

逐项验证：

| $x$ | $\sigma^2(x)=x+2$ | 传统"我克" | 对吗 |
|----|----|----|----|
| 木(0) | 土(2) | 木克土 | ✓ |
| 土(2) | 水(4) | 土克水 | ✓ |
| 水(4) | 火(1) | 水克火 | ✓ |
| 火(1) | 金(3) | 火克金 | ✓ |
| 金(3) | 木(0) | 金克木 | ✓ |

全部命中。因此

$$
\boxed{\;\text{相克} = (\text{相生})^2 = \sigma^2\;}
$$

**含义**：相生和相克不是两套各自发明的规则，而是**同一个循环群的两次幂**。
传统把它们讲成两种独立关系，代数揭示了它们同源。这是整个体系里最干净的一处对称性。

## 4. 生克体系 = C₅

由 $\sigma$ 生成的群 $\langle\sigma\rangle = \{e,\sigma,\sigma^2,\sigma^3,\sigma^4\}$ 同构于 $C_5 \cong \mathbb{Z}_5$：

$$
\langle\sigma\rangle \cong \mathbb{Z}_5
$$

五种幂 $\sigma^k$ 的语义恰好对应五行的四种基本关系 + 比和：

| 幂 $k$ | $\sigma^k(x) = x+k$ | 关系名 | 含义 |
|----|----|----|----|
| 0 | $x$ | 比和 | 同我（同类） |
| 1 | $x+1$ | 我生 | 我所生（食伤） |
| 2 | $x+2$ | **我克** | 我所克（妻财） |
| 3 | $x+3$ | 克我 | 克我者（官杀） |
| 4 | $x+4$ | 生我 | 生我者（印星） |

**五行的全部横向关系，被一个差分 $k = y - x \pmod 5$ 一次性编码。**

## 5. 关系矩阵（差分表）

任意两行 $x, y \in \mathbb{Z}_5$ 的生克关系，只取决于差值 $d = y - x \pmod 5$：

$$
\mathrm{rel}(x, y) = (y - x) \bmod 5 \;\in\; \{0,1,2,3,4\}
$$

这是一张**循环差分表 / circulant matrix**，是后续十神、合化等算子的基础：

```
        y: 木 火 土 金 水
x
木(0)    0  1  2  3  4
火(1)    4  0  1  2  3
土(2)    3  4  0  1  2
金(3)    2  3  4  0  1
水(4)    1  2  3  4  0
```

> 对称性：表中 $(x,y)$ 与 $(y,x)$ 的值互补（和为 0 或 5）。"我克"与"克我"、"我生"与"生我"
> 成对出现，正是 $d \leftrightarrow -d$。

## 6. 反向关系算子

由 $\sigma$ 可派生：

- **被生**：$\sigma^{-1}(x) = x - 1$（谁生我）
- **被克**：$\sigma^{-2}(x) = x - 2 = x + 3$（谁克我）

无需新公理，都是 $\sigma$ 的幂。

## 7. 与天干的衔接（回看 03）

天干的五行归属 $\tau: \mathbb{Z}_{10}\to\mathbb{Z}_5,\ \tau(a)=\lfloor a/2\rfloor$ 把干的世界接入五行世界。
于是**任意两干的生克关系**可纯代数算出：

$$
\mathrm{rel}(\text{干}_a,\ \text{干}_b) = \big(\lfloor b/2\rfloor - \lfloor a/2\rfloor\big) \bmod 5
$$

这正是"十神"定义的内核（见 [07-operators](07-operators.md)）：只需再叠一层阴阳奇偶即可得到 10 类。

## 8. 五行的扩展映射（应用层，不进内核）

五行还可以映射到五方、五季、五色、五脏、五志…… 这些都是 $E \to S$ 的映射，**属消费者/算子层**，
不污染内核。列举备查：

| 维度 | 木 | 火 | 土 | 金 | 水 |
|----|----|----|----|----|----|
| 方位 | 东 | 南 | 中 | 西 | 北 |
| 季节 | 春 | 夏 | 长夏 | 秋 | 冬 |
| 五脏 | 肝 | 心 | 脾 | 肺 | 肾 |
| 五志 | 怒 | 喜 | 思 | 悲 | 恐 |

## 9. 本章小结

| 对象 | 代数身份 |
|----|----|
| 五行集 $E$ | $\mathbb{Z}_5$ |
| 相生 $\sigma$ | $x\mapsto x+1$，5-循环 |
| 相克 | $\sigma^2$，**生克的代数统一** |
| 生克体系 $\langle\sigma\rangle$ | $C_5\cong\mathbb{Z}_5$ |
| 任意两行关系 | 差分 $d=y-x\pmod5$ |
| 干→五行 $\tau$ | $a\mapsto\lfloor a/2\rfloor$ |
