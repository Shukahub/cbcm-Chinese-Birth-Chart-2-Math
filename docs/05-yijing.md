# 05 · 阴阳·八卦·六十四卦

> 本章把易经的符号系统写成 $\mathbb{F}_2$ 上的向量空间。莱布尼茨三百年前就看出来：六十四卦就是二进制。

---

## 0. 传统说法（古法原典）

> 本章先按古人自己的话讲一遍，再做代数翻译（文末小结即对照表）。

- **太极生两仪**：阴、阳。符号为阴爻 `--`、阳爻 `—`。《周易·系辞》："易有太极，是生两仪，两仪生四象，四象生八卦。"
- **八卦**：每卦三爻，共八卦——乾 ☰、兑 ☱、离 ☲、震 ☳、巽 ☴、坎 ☵、艮 ☶、坤 ☷。象天地风雷水火山泽。
- **六十四卦**：八卦相重（上卦+下卦），每卦六爻，共 $8\times8=64$ 卦，即《周易》全经。
- **卦变**：
  - **变**：某一爻阴阳互换；
  - **错卦**：六爻全部阴阳互换（如乾 ☰ ↔ 坤 ☷）；
  - **综卦**：整个卦上下颠倒；
  - **互卦**：取二三四五爻重新组成上下卦。
- **两套卦序**：**先天八卦**（伏羲，乾坤定南北，乾兑离震巽坎艮坤）与**后天八卦**（文王，离坎定南北）。

> 关键观察：古人"两仪—八卦—六十四卦"的倍增过程，恰是 $2\to2^3\to2^6$ 的幂次——下面直接写成 $\mathbb{F}_2$ 上的向量空间。

## 1. 阴阳 = F₂

定义阴阳集 $Y = \mathbb{F}_2 = \{0,1\}$：

```
阴爻 = 0    (--)
阳爻 = 1    (—)
```

- 加法 = XOR：$0+0=0,\ 0+1=1,\ 1+1=0$。
- 含义：**两爻相加 = 翻转**（阳变阴、阴变阳）。这就是"变"。
- $\mathbb{F}_2$ 是域（可加可乘），比单纯的 $\mathbb{Z}_2$ 群更强——能做线性代数。

## 2. 爻、卦 = 序列

- **爻**：一个 $\mathbb{F}_2$ 元。
- **卦**：有限长度的 $\mathbb{F}_2$ 序列。
- 传统读法：卦从**下往上**画，从**下往上**编号（初爻、二爻、…、上爻）。
  约定序列下标 0 = 初爻（最下），与"自下而上"一致。

## 3. 八卦 = (F₂)³

三个爻的卦 = 长度 3 的 $\mathbb{F}_2$ 序列 = 集合 $\mathbb{F}_2^3$，共 $2^3=8$ 个元素：

| 卦 | 序列 (下→上) | 二进制 | 名 |
|----|----|----|----|
| ☷ | 0,0,0 | 000 | 坤 |
| ☳ | 1,0,0 | 001 | 震 |
| ☵ | 0,1,0 | 010 | 坎 |
| ☱ | 1,1,0 | 011 | 兑 |
| ☶ | 0,0,1 | 100 | 艮 |
| ☲ | 1,0,1 | 101 | 离 |
| ☴ | 0,1,1 | 110 | 巽 |
| ☰ | 1,1,1 | 111 | 乾 |

（按二进制大小排序；传统"先天八卦序"是另一种标号，见 §5。）

$\mathbb{F}_2^3$ 是 $\mathbb{F}_2$ 上的 3 维向量空间。**任意两卦可相加（逐位 XOR）= 对应位变化**。

## 4. 六十四卦 = (F₂)⁶

六个爻的卦 = $\mathbb{F}_2^6$，共 $2^6=64$ 元，是 $\mathbb{F}_2$ 上 6 维向量空间。

- 一个六十四卦 = 一个 6 位二进制数（莱布尼iz的发现）。
- 也可视为两个八卦（下卦 + 上卦）的拼接：$\mathbb{F}_2^6 \cong \mathbb{F}_2^3 \times \mathbb{F}_2^3$。

## 5. 传统卦变 = 线性变换

把卦当向量后，传统"卦变"操作全是**简单的线性运算**：

| 传统术语 | 代数操作 | 公式 |
|----|----|----|
| 变某一爻 | 加单位向量 | $x \mapsto x + e_i$ |
| **错卦**（阴阳全反） | 加全 1 向量 | $x \mapsto x + \mathbf{1}$ |
| **综卦**（上下颠倒） | 反转下标 | $x_i \mapsto x_{5-i}$（置换矩阵） |
| **互卦**（取 2~5 爻） | 取子向量重组 | 投影 + 重排 |
| 动爻成"之卦" | 叠加若干 $e_i$ | $x \mapsto x + d,\ d\in\mathbb{F}_2^6$ |

这些都属 $\mathbb{F}_2^6$ 上的**仿射变换 / 置换**，可矩阵化、可程序化。

## 6. 先天 / 后天八卦序 = 两套标号

八卦本身是集合 $\mathbb{F}_2^3$（无序）。"先天八卦序"（乾兑离震巽坎艮坤）和"后天八卦序"
（离坤兑乾坎艮震巽）是这个集合上的**两种排列**——即 $\mathbb{F}_2^3$ 上的两个双射。

> 这些"序"承载的是应用层（方位、节气、风水）的约定，**不进内核**。
> 内核只提供集合 $\mathbb{F}_2^3$ 和线性运算；序作为查表/标号放在算子层。

## 7. 与五行的衔接

八卦也带五行（乾兑金、震巽木、坎水、离火、坤艮土），是一个映射 $\beta: \mathbb{F}_2^3 \to \mathbb{Z}_5$，
把易经世界接入 [04](04-wuxing.md) 的五行代数。具体 $\beta$ 在算子层定义。

## 8. 本章小结

| 对象 | 代数身份 |
|----|----|
| 阴阳 | $\mathbb{F}_2$（域） |
| 八卦 | $\mathbb{F}_2^3$（3 维向量空间） |
| 六十四卦 | $\mathbb{F}_2^6$（6 维向量空间） |
| 变爻/错/综/互 | 仿射变换、置换（线性） |
| 先天/后天序 | 集合上的双射（应用层） |
| 卦→五行 $\beta$ | 映射（应用层） |

至此内核三大块齐了：**干支（Z₆₀）、五行（C₅）、易经（F₂ⁿ）**。
