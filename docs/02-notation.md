# 02 · 数学符号约定

本文统一全项目的符号。**后面所有文档默认遵守这里的约定，不再重复说明。**

## 1. 代数记号

| 记号 | 含义 |
|----|----|
| $\mathbb{Z}_n$ | 整数模 $n$ 的加法群（同时是环），元素 $\{0,1,\dots,n-1\}$，运算为模 $n$ 加/乘 |
| $\mathbb{F}_2$ | 二元域 $\{0,1\}$，加法为 XOR，乘法为 AND |
| $\mathbb{Z}_2^k$ | $\mathbb{F}_2$ 上的 $k$ 维向量空间，共 $2^k$ 个元素 |
| $C_n$ | $n$ 阶循环群，同构于 $\mathbb{Z}_n$ |
| $\sigma$ | （五行章节）相生循环置换 |
| $\cong$ | 同构 |
| $\langle g\rangle$ | 元素 $g$ 生成的循环子群 |

## 2. 索引约定（重要，避免踩坑）

传统干支**从 1 开始数**：甲=第1、子=第1。但群论里**从 0 开始**运算才干净（模运算对齐）。

**本项目一律内部用 0-indexed，显示时再翻译回名字。**

- 天干：甲=0, 乙=1, …, 癸=9 ∈ $\mathbb{Z}_{10}$
- 地支：子=0, 丑=1, …, 亥=11 ∈ $\mathbb{Z}_{12}$
- 五行：木=0, 火=1, 土=2, 金=3, 水=4 ∈ $\mathbb{Z}_5$
- 阴阳：阴=0, 阳=1 ∈ $\mathbb{F}_2$
- 爻：阴爻=0, 阳爻=1 ∈ $\mathbb{F}_2$

> 这样选的好处：模运算、奇偶（阴阳）、差分（生克）全部直接落在数字上，无需反复 ±1。

## 3. 奇偶 = 阴阳

0-indexed 下，**序号的奇偶性直接就是阴阳**：

- 天干：偶序号 = 阳（甲丙戊庚壬），奇序号 = 阴（乙丁己辛癸）
- 地支：偶序号 = 阳（子寅辰午申戌），奇序号 = 阴（丑卯巳未酉亥）

记 $\pi(x) = x \bmod 2$ 为阴阳算子。这在排盘、生克、六十甲子里反复用到。

## 4. 名称表

代数对象只认数字；要显示成中文，靠**名称表**（纯查表，不参与运算）：

```
STEM[10]   = [甲, 乙, 丙, 丁, 戊, 己, 庚, 辛, 壬, 癸]      # Z_10
BRANCH[12] = [子, 丑, 寅, 卯, 辰, 巳, 午, 未, 申, 酉, 戌, 亥] # Z_12
WUXING[5]  = [木, 火, 土, 金, 水]                            # Z_5
```

## 5. 约定口诀

> **内部全用数，显示才查表；0 起 mod n，奇偶即阴阳。**
