# 08 · 预测消费者（占位 · 提纲）

> **状态：提纲。** 这是把"逻辑"接到"经验"的一层——按 [01](01-philosophy.md) 的边界，**所有经验断言都只能活在这一层**，绝不污染内核。

## 定位

预测消费者 = 读"命盘状态 + 算子标注"，输出对一个**具体维度**（性格、情志、健康…）的刻画。

```
命盘状态 G⁴  ──┐
              ├─→  消费者  ─→  性格画像 / 情志倾向 / 健康提示 ...
算子标注      ──┘
```

## 两条路线（先传统，后数据）

### 路线 A：录传统断法（先做）
- 把传统命理典籍里"某十神旺→某性格"等规则，编码成确定映射。
- 产出：可解释、可复现的"按古法"画像。
- 价值：先有一致的参照系，再谈是否准。

### 路线 B：数据驱动拟合（后做）
- 把状态/标注当特征，性格问卷/行为数据当标签，做统计学习。
- 价值：检验断法是否经得起数据，并发现新的关联。
- 前提：需要数据集 + 已稳定的内核特征定义——这正是前几章在做的事。

## 本项目的主消费目标

按规划，**性格 / 情志**是主力应用。拟维度：

- 大五人格 / MBTI 等现代心理学量表作为可对齐的标签集。
- 五志（怒喜思悲恐，见 [04 §8](04-wuxing.md)）作为传统侧对应。
- 探索两侧映射关系（先假设性、后验证性）。

## 边界提醒

- 任何"性格结论"都标注为**消费者输出**，内核不背书。
- 用户应能切换/替换消费者，得到不同结论；内核特征不变。
